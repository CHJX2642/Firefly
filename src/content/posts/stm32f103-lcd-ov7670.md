## 🚀 项目简介

这是一个基于 **STM32F103ZET6** 的嵌入式视觉系统，实现了 **OV7670 摄像头**实时图像采集与 **ILI9341 LCD** 显示。

**Gitee 地址：** [260526_STM32F103ZET6_LCD_OV7670](https://gitee.com/chjx2642/260526_-stm32-f103-zet6_-lcd_-ov7670)

### 为什么要做这个项目？

在嵌入式开发中，图像采集与显示是一个经典的应用场景：

1. **工业视觉检测**：产品缺陷检测、尺寸测量
2. **智能家居**：门禁人脸识别、监控摄像头
3. **机器人视觉**：避障、目标追踪
4. **学习入门**：理解图像采集、帧同步、显示驱动等原理

## ✨ 功能特性

- **实时图像采集**：OV7670 摄像头 QVGA(320x240) 模式
- **FIFO 帧缓冲**：AL422B 384KB FIFO 缓冲一帧完整图像
- **LCD 实时显示**：ILI9341 2.8寸 TFT，16位色，FSMC 接口
- **双 VSYNC 帧同步**：消除图像撕裂，帧率约 35fps
- **SCCB 配置**：软件模拟 SCCB 协议配置 OV7670 寄存器

## 🏗️ 硬件平台

| 模块 | 型号 | 说明 |
|------|------|------|
| 主控 | STM32F103ZET6 | Cortex-M3 @ 72MHz, 512KB Flash, 64KB SRAM |
| 摄像头 | OV7670 | 30万像素, 最大 VGA(640x480), 本项目使用 QVGA(320x240) |
| 帧缓冲 | AL422B | 384KB FIFO, 缓冲一帧 QVGA RGB565 数据 |

## 🔌 引脚连接

### OV7670 摄像头

| 信号 | 引脚 | 说明 |
|------|------|------|
| XCLK | PA6 | TIM3_CH1 PWM 输出 8MHz |
| VSYNC | PA8 | 场同步信号, 上升沿触发中断 |
| SCL | PD3 | SCCB 时钟线 (开漏) |
| SDA | PG13 | SCCB 数据线 (开漏) |
| D0-D7 | PC0-PC7 | 8位并行数据输入 |

### AL422B FIFO 控制

| 信号 | 引脚 | 说明 |
|------|------|------|
| WEN | PB3 | 写使能 (拉低暂停写入) |
| RCLK | PB4 | 读时钟 (上升沿读数据) |
| WRST | PD6 | 写指针复位 |
| RRST | PG14 | 读指针复位 |
| OE | PG15 | 数据输出使能 |

### ILI9341 LCD (FSMC Bank1 NE4)

| 信号 | 引脚 | 说明 |
|------|------|------|
| D0-D15 | PD14-PD15, PD0-PD1, PE7-PE15, PD8-PD10 | 16位数据总线 |
| RS | PD11 | 命令/数据选择 (FSMC A16) |
| CS | PG12 | 片选 (FSMC NE4) |
| WR | PD5 | 写使能 (FSMC NWE) |
| RD | PD4 | 读使能 (FSMC NOE) |
| RST | PG11 | 硬件复位 |
| BL | PB0 | 背光控制 (PWM) |

## 🔧 核心代码解析

### 1. 双 VSYNC 帧同步机制

这是项目的核心算法，解决了图像撕裂问题：

```c
// main.c - VSYNC 中断处理
volatile uint8_t vsync_count = 0;
volatile uint8_t frame_ready = 0;

void EXTI9_5_IRQHandler(void) {
    if (__HAL_GPIO_EXTI_GET_IT(VSYNC_PIN)) {
        __HAL_GPIO_EXTI_CLEAR_IT(VSYNC_PIN);
        
        vsync_count++;
        
        // 隔次 WRST 机制
        if (vsync_count % 2 == 1) {
            // 奇数次 VSYNC：复位写指针，开始写入新帧
            FIFO_WRST_LOW();   // 拉低 WRST
            delay_us(1);
            FIFO_WRST_HIGH();  // 释放 WRST
            FIFO_WEN_ENABLE(); // 使能写入
        } else {
            // 偶数次 VSYNC：跳过 WRST，MCU 安全读取
            FIFO_WEN_DISABLE(); // 暂停写入
            frame_ready = 1;    // 标记帧就绪
        }
    }
}
```

**原理说明：**
1. OV7670 输出 ~30fps VGA 数据，MCU 读取 QVGA 窗口
2. MCU 读 FIFO 速度 (~12MB/s) 快于摄像头写入速度 (~8MB/s)
3. 奇数次 VSYNC 复位写指针，开始写入新帧
4. 偶数次 VSYNC 跳过，FIFO 中前一帧完整，MCU 安全读取
5. 每 2 次 VSYNC 显示一帧，实际帧率约 35fps

### 2. OV7670 SCCB 驱动

SCCB 协议兼容 I2C，用于配置 OV7670 寄存器：

```c
// ov7670.c - SCCB 写操作
#define SCL_HIGH()  HAL_GPIO_WritePin(SCL_PORT, SCL_PIN, GPIO_PIN_SET)
#define SCL_LOW()   HAL_GPIO_WritePin(SCL_PORT, SCL_PIN, GPIO_PIN_RESET)
#define SDA_HIGH()  HAL_GPIO_WritePin(SDA_PORT, SDA_PIN, GPIO_PIN_SET)
#define SDA_LOW()   HAL_GPIO_WritePin(SDA_PORT, SDA_PIN, GPIO_PIN_RESET)
#define SDA_READ()  HAL_GPIO_ReadPin(SDA_PORT, SDA_PIN)

void sccb_start(void) {
    SDA_HIGH();
    SCL_HIGH();
    delay_us(2);
    SDA_LOW();
    delay_us(2);
    SCL_LOW();
}

void sccb_stop(void) {
    SDA_LOW();
    SCL_HIGH();
    delay_us(2);
    SDA_HIGH();
    delay_us(2);
}

uint8_t sccb_write_byte(uint8_t data) {
    uint8_t i;
    for (i = 0; i < 8; i++) {
        if (data & 0x80) SDA_HIGH();
        else SDA_LOW();
        data <<= 1;
        SCL_HIGH();
        delay_us(2);
        SCL_LOW();
    }
    // 读取 ACK
    SDA_HIGH();
    SCL_HIGH();
    delay_us(2);
    uint8_t ack = SDA_READ();
    SCL_LOW();
    return ack;
}

// 写 OV7670 寄存器
uint8_t ov7670_write_reg(uint8_t reg, uint8_t val) {
    sccb_start();
    sccb_write_byte(0x42);  // 设备地址 + 写
    sccb_write_byte(reg);   // 寄存器地址
    sccb_write_byte(val);   // 数据
    sccb_stop();
    return 0;
}
```

### 3. OV7670 寄存器配置

配置 OV7670 工作在 QVGA 模式：

```c
// ov7670.c - 初始化配置
void ov7670_init(void) {
    // 复位 OV7670
    ov7670_write_reg(0x12, 0x80);  // COM7: 软件复位
    delay_ms(10);
    
    // 配置输出格式
    ov7670_write_reg(0x12, 0x14);  // COM7: QVGA, RGB565
    ov7670_write_reg(0x11, 0x80);  // CLKRC: 内部时钟
    ov7670_write_reg(0x0C, 0x00);  // COM3: 缩放使能
    ov7670_write_reg(0x3E, 0x13);  // COM14: 像素时钟, RGB565
    
    // 配置时序
    ov7670_write_reg(0x17, 0x16);  // HSTART: 水平起始
    ov7670_write_reg(0x18, 0x04);  // HSTOP: 水平停止
    ov7670_write_reg(0x19, 0x02);  // VSTART: 垂直起始
    ov7670_write_reg(0x1A, 0x7A);  // VSTOP: 垂直停止
    ov7670_write_reg(0x32, 0x80);  // HREF: 行参考
    
    // 配置增益和曝光
    ov7670_write_reg(0x13, 0x8F);  // COM8: 自动增益/曝光
    ov7670_write_reg(0x00, 0x00);  // GAIN: 增益
    ov7670_write_reg(0x10, 0x40);  // AEC: 自动曝光控制
    
    // 配置色彩
    ov7670_write_reg(0x14, 0x49);  // COM9: 增益上限
    ov7670_write_reg(0xB1, 0x00);  // MTX1: 色彩矩阵
    ov7670_write_reg(0xB2, 0x77);  // MTX2
    ov7670_write_reg(0xB3, 0x77);  // MTX3
    ov7670_write_reg(0xB4, 0x00);  // MTX4
    ov7670_write_reg(0xB5, 0x00);  // MTX5
    ov7670_write_reg(0xB6, 0x3D);  // MTX6
    ov7670_write_reg(0xB7, 0x3D);  // MTX7
    ov7670_write_reg(0xB8, 0x3D);  // MTX8
    ov7670_write_reg(0xB9, 0x3D);  // MTX9
    
    // 配置伽马校正
    ov7670_write_reg(0xBA, 0x00);  // GFIX: 伽马
    ov7670_write_reg(0xBB, 0x80);  // GGAIN: 伽马增益
    
    printf("OV7670 初始化完成\r\n");
}
```

### 4. FIFO 读取与 LCD 显示

从 FIFO 读取图像数据并显示到 LCD：

```c
// main.c - 图像采集与显示
void capture_and_display(void) {
    uint16_t x, y;
    uint8_t rgb[2];
    
    // 检查帧就绪
    if (!frame_ready) return;
    frame_ready = 0;
    
    // 开始读取 FIFO
    FIFO_OE_LOW();      // 使能输出
    FIFO_RCLK_LOW();    // 读时钟拉低
    delay_us(1);
    
    // 设置 LCD 窗口
    LCD_SetWindow(0, 0, 319, 239);
    
    // 读取 QVGA 数据 (320x240)
    for (y = 0; y < 240; y++) {
        for (x = 0; x < 320; x++) {
            // 读取 RGB565 高字节
            FIFO_RCLK_HIGH();
            delay_us(1);
            rgb[0] = FIFO_DATA_READ();  // D0-D7
            FIFO_RCLK_LOW();
            delay_us(1);
            
            // 读取 RGB565 低字节
            FIFO_RCLK_HIGH();
            delay_us(1);
            rgb[1] = FIFO_DATA_READ();  // D0-D7
            FIFO_RCLK_LOW();
            delay_us(1);
            
            // 写入 LCD
            LCD_WriteData((rgb[1] << 8) | rgb[0]);
        }
    }
    
    // 停止读取
    FIFO_OE_HIGH();     // 禁用输出
    FIFO_RRST_LOW();    // 复位读指针
    delay_us(1);
    FIFO_RRST_HIGH();
}
```

### 5. LCD 驱动（FSMC）

使用 FSMC 接口驱动 LCD，速度更快：

```c
// lcd.c - FSMC LCD 驱动
#define LCD_BASE  ((uint32_t)(0x6C000000 | 0x0000007E))  // FSMC Bank1 NE4

void LCD_WriteCommand(uint16_t cmd) {
    *(__IO uint16_t *)LCD_BASE = cmd;
}

void LCD_WriteData(uint16_t data) {
    *(__IO uint16_t *)(LCD_BASE | (1 << 17)) = data;  // RS=1
}

void LCD_SetWindow(uint16_t x0, uint16_t y0, uint16_t x1, uint16_t y1) {
    LCD_WriteCommand(0x2A);  // 列地址设置
    LCD_WriteData(x0 >> 8);
    LCD_WriteData(x0 & 0xFF);
    LCD_WriteData(x1 >> 8);
    LCD_WriteData(x1 & 0xFF);
    
    LCD_WriteCommand(0x2B);  // 行地址设置
    LCD_WriteData(y0 >> 8);
    LCD_WriteData(y0 & 0xFF);
    LCD_WriteData(y1 >> 8);
    LCD_WriteData(y1 & 0xFF);
    
    LCD_WriteCommand(0x2C);  // 写入显存
}
```
